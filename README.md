# 🎯 Makhraj – Quran Recognizer (Automatic Speech Recognition System)

## 📌 Overview
**Makhraj (Quran Recognizer)** is an AI-powered Automatic Speech Recognition (ASR) system designed to analyze Quran recitation and provide **real-time pronunciation feedback**.

The system leverages **Transformer-based models (Whisper)** fine-tuned specifically for Quranic Arabic, enabling accurate recognition and evaluation of recitation quality.

## 🚀 Features
- 🎙️ Real-time speech recognition for Quran recitation  
- 🧠 Fine-tuned Transformer model (Whisper)  
- ⚡ Real-time feedback using WebSockets  
- ☁️ Cloud deployment on Microsoft Azure  
- 📊 Accurate pronunciation analysis

## 📊 How It Works
1. User recites Quran through the application  
2. Audio is captured and preprocessed  
3. The **fine-tuned Whisper model** transcribes the speech  
4. The system analyzes pronunciation accuracy  
5. Real-time feedback is sent back to the user
